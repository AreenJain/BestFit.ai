<h1>🚀 BestFit.AI - AI-Powered Job Search Assistant</h1>
<h2><a href="https://bestfit-ai.streamlit.app/" target="_blank">Project Link</a></h2>

<p>
    BestFit.AI is an AI-powered job search assistant that helps users find relevant job opportunities based on their 
    <strong>skills</strong> and <strong>preferred location</strong>. It leverages <strong>LLMs (Large Language Models)</strong> 
    to extract skills from user input and fetches real-time job listings.
</p>

<h2>🔹 Features</h2>
<ul>
    <li>✅ <strong>Skill-Based Job Search</strong> – Extracts skills from user input and finds relevant jobs.</li>
    <li>✅ <strong>Location-Specific Jobs</strong> – Users can specify a location to filter job results.</li>
    <li>✅ <strong>Real-Time Job Fetching</strong> – Uses an external API to get up-to-date job listings.</li>
    <li>✅ <strong>Streamlit UI (Optional)</strong> – Can be integrated into a simple UI for user interaction.</li>
</ul>

<h2>🛠️ Tech Stack</h2>
<ul>
    <li><strong>Python</strong> 🐍</li>
    <li><strong>Requests</strong> (for API calls)</li>
    <li><strong>Streamlit</strong> (for UI, if applicable)</li>
    <li><strong>Natural Language Processing (NLP)</strong> (for skill extraction)</li>
</ul>

<h2>📌 How It Works</h2>
<ol>
    <li>The user enters a <strong>job title or description</strong>.</li>
    <li>The AI extracts <strong>relevant skills</strong> using an <strong>LLM</strong>.</li>
    <li>The user selects a <strong>location</strong> for job search.</li>
    <li>The system queries the <strong>job search API</strong> to fetch job listings.</li>
    <li>Results are displayed, including:
        <ul>
            <li><strong>Job Title</strong></li>
            <li><strong>Company Name</strong></li>
            <li><strong>Location</strong></li>
            <li><strong>Application Link</strong></li>
            <li><strong>Job Description</strong></li>
        </ul>
    </li>
</ol>
